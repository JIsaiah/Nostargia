<h2>プロジェクト概要</h2>
<p>本サービスは、デジタル社会における過度な承認欲求や他者評価への依存による精神的疲弊を軽減し、ユーザーが自分自身と静かに向き合える「個人的な記録空間」を提供することを目的とする。
「いいね」や数値評価を排除した設計により、他人の視線から解放された安心できる場所を実現する。
<br>
また、日常の成功・失敗や「何もしなかった一日」も含め、すべての経験を価値ある人生の一部として可視化することで、ユーザーが自身の歩みを肯定的に捉え直し、自己肯定感を回復・維持するきっかけを創出する。
</p>
<br>
<h2>想定ユースケース</h2>
<p>仕事帰りに疲れていてSNSを見る気にならない。今日は面白い出来事があったが他人に見せるほどないけれど残しておきたい気持ちはある。寝る前にノスタージアでそれを自分のそらに打ち上げていつか自分の励ましになると思って心が落ち着く。その日の振り返りをしながら星座になりかけている星を眺めて当時の思い出を思い浮かぶ。<p>
